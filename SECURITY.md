# Security Policy

## Supported Versions

This is a personal project. There is no formal security SLA or long-term support window.

## Reporting a Vulnerability

- For security-sensitive issues, contact: `3230105930@zju.edu.cn`
- For non-sensitive bugs, open a public issue: <https://github.com/JiaJunDeng5930/TypeVoice/issues>

When reporting, include:
- Affected version or commit
- Reproduction steps
- Potential impact
- Suggested fix (optional)
